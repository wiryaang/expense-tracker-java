# Expense Tracker (Java)

A console-based expense tracker built in Java with SQLite database integration.

## Features
- Add, view, and delete expenses
- Calculate total expenses
- View spending by category
- Persistent storage using SQLite

## Technologies Used
- Java
- SQLite (JDBC)

## Project Structure
- `model` → data classes
- `service` → business logic
- `ui` → user interface (console)
- `db` → database connection

## How to Run
1. Clone the repository
2. Open in NetBeans or any Java IDE
3. Run `Main.java`

## Status
Completed basic version with database persistence. Future improvements may include GUI and advanced filtering.
